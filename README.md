# 🧠 ML Model Dashboard using Streamlit

## 📌 Project Overview

This project is an **interactive Machine Learning Dashboard** built using **Streamlit**. It allows users to input feature values and simulate predictions in real time.

The dashboard provides:

* User-friendly input controls
* Real-time prediction output
* Confidence score display
* Visual representation of feature values

This project demonstrates how machine learning models can be deployed through a simple and interactive web interface.

---

## 🎯 Objectives

* Build an interactive dashboard for ML predictions
* Allow users to input feature values dynamically
* Display prediction results instantly
* Visualize input data using charts
* Demonstrate model deployment using Streamlit

---

## ⚙️ Technologies Used

* Python
* Streamlit
* NumPy
* Pandas
* Matplotlib

## 🚀 How to Run This Project

### Step 1 — Clone Repository

```bash
git clone https://github.com/Sharanmithra/ml-model-dashboard.git
cd ml-model-dashboard
```

### Step 2 — Install Dependencies

```bash
pip install -r requirements.txt
```

### Step 3 — Run Application

```bash
streamlit run app.py
```
